话不多说，先上图

![UI](http://139.196.214.241:8093/cdn/screenshot/movie%20v2.0.jpg)

## 感谢与支持
- 豆瓣api https://developers.douban.com/
- 百度地图 http://lbsyun.baidu.com/

## 新版本内容

- 将搜索从页面转移到首页面的搜索框
- 增加了电影与人物的浏览历史、收藏功能
- 添加“我的页面”（重头戏）
- 添加摇一摇、相册模块
- 添加设置功能，可清除缓存、编辑个人资料、查看手机信息、更新地理位置
- 添加关于页面，里面列举除了项目中所有使用到的api
